# MatchPick Quiniela API

![Laravel](https://img.shields.io/badge/Laravel-12.x-FF2D20?style=flat&logo=laravel&logoColor=white)
![PHP](https://img.shields.io/badge/PHP-8.3-777BB4?style=flat&logo=php&logoColor=white)

![Docker](https://img.shields.io/badge/Docker-ready-2496ED?style=flat&logo=docker&logoColor=white)

REST API for the MatchPick Quinielas platform. Manage football prediction pools (quinielas) for the FIFA World Cup 2026 and other tournaments. Authentication via Laravel Sanctum (Bearer tokens) and Google OAuth via Socialite.

---

## Stack

- **Runtime:** PHP 8.3, Laravel 12
- **Database:** PostgreSQL 16
- **Auth:** Laravel Sanctum (Bearer tokens) + Google OAuth (Socialite)
- **Infrastructure:** Docker, Nginx (Alpine), php-fpm

---

## Quick Start with Docker

```bash
# 1. Clone and setup environment
cp .env.example .env

# 2. Build and start all services
docker compose up -d --build

# 3. Install dependencies and initialize the application
docker compose exec app composer install
docker compose exec app php artisan key:generate
docker compose exec app php artisan migrate
docker compose exec app php artisan db:seed

# 4. API is ready at
# http://localhost:8000
```

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `APP_KEY` | Laravel app encryption key (generated by `artisan key:generate`) | — |
| `DB_HOST` | PostgreSQL host | `db` |
| `DB_DATABASE` | Database name | `quiniela` |
| `DB_USERNAME` | Database user | `quiniela` |
| `DB_PASSWORD` | Database password | `secret` |
| `SANCTUM_STATEFUL_DOMAINS` | Domains for stateful SPA auth | `localhost:3000` |
| `FRONTEND_URL` | Frontend URL for OAuth redirects | `http://localhost:3000` |
| `GOOGLE_CLIENT_ID` | Google OAuth Client ID | — |
| `GOOGLE_CLIENT_SECRET` | Google OAuth Client Secret | — |
| `GOOGLE_REDIRECT_URI` | Google OAuth callback URL | `http://localhost:8000/api/auth/google/callback` |

## API Endpoints

### Authentication

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `POST` | `/api/auth/register` | Register a new user | Public |
| `POST` | `/api/auth/login` | Login with email/password | Public |
| `POST` | `/api/auth/logout` | Revoke current token | Bearer |
| `GET` | `/api/auth/me` | Get authenticated user | Bearer |
| `GET` | `/api/auth/google/redirect` | Get Google OAuth URL | Public |
| `GET` | `/api/auth/google/callback` | Handle Google OAuth callback | Public |

### Tournaments

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `GET` | `/api/tournaments` | List active tournaments | Public |
| `GET` | `/api/tournaments/{slug}` | Get tournament details | Public |
| `GET` | `/api/tournaments/{slug}/matches` | List matches by round | Public |

### Quinielas

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `GET` | `/api/quinielas` | List user's quinielas | Bearer |
| `POST` | `/api/quinielas` | Create a new quiniela | Bearer |
| `GET` | `/api/quinielas/{slug}` | Get quiniela details | Bearer |
| `PUT` | `/api/quinielas/{slug}` | Update quiniela (admin only) | Bearer |
| `DELETE` | `/api/quinielas/{slug}` | Delete quiniela (admin only) | Bearer |
| `GET` | `/api/quinielas/{slug}/matches` | Matches with user's predictions | Bearer |
| `GET` | `/api/quinielas/{slug}/standings` | Leaderboard | Bearer |

### Predictions

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `POST` | `/api/quinielas/{slug}/predictions` | Bulk upsert predictions | Bearer |
| `GET` | `/api/quinielas/{slug}/predictions/{matchId}` | View all predictions for a match | Bearer |

### Invitations

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `POST` | `/api/quinielas/{slug}/invitations` | Create invitation (admin only) | Bearer |
| `GET` | `/api/invitations/{token}` | View invitation details | Public |
| `POST` | `/api/invitations/{token}/accept` | Accept invitation | Bearer |

### Admin

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| `POST` | `/api/admin/matches/{match}/results` | Submit match result | Bearer + is_admin |

## Scoring System

| Result | Points |
|--------|--------|
| Exact score (e.g. 2-1 predicted, 2-1 actual) | 3 pts |
| Correct winner/draw (e.g. home win predicted, home win actual) | 1 pt |
| Wrong result | 0 pts |

## Project Structure

```
app/
  Http/
    Controllers/    # Auth, Tournament, Quiniela, Prediction, Standing, Invitation, MatchResult
    Requests/       # Form request validation
    Resources/      # API response transformers
  Jobs/             # CalculateScoresJob (async score calculation)
  Models/           # Eloquent models
  Providers/        # AppServiceProvider (rate limiting)
  Services/         # ScoringService
database/
  migrations/       # 14 migration files
  seeders/          # World Cup 2026 seed data (48 countries, teams, tournament, rounds, 72 group matches)
docker/
  nginx/            # Nginx config
routes/
  api.php           # All API routes
```
